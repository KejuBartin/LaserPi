# LaserPi
LaserPi is a software to turn a projector into a showlaser in combination with a hazer or a fog machine using a Raspberry Pi.

## Requirements
- `pygame-ce`

# Additional information
LaserPi is currently under development and nowhere near finished. When executing the `shapeTest.py` file, a fullscreen window with shapes will appear on the main display. More functionality will be added over time.
